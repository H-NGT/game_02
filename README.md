# Sketch Armory 3D

A small browser game inspired by 3D draw-and-fight arena games. Draw a weapon,
attack, and try to push the opponent off the platform.

This project does not copy the original app's name, art, assets, code, or
branding.

## Run

```bash
npm install
npm run dev
```

## Deploy

Pushes to `main` deploy the built game to GitHub Pages.
